L'Oréal Performance Analysis Dashboard
Overview

This Power BI project analyzes the performance of L'Oréal products using customer reviews, ratings, and product category data. The dashboard provides insights into product popularity, customer satisfaction, and category performance to support business decision-making.

Project Objectives
Analyze product performance across categories.
Evaluate customer ratings and reviews.
Identify top-performing products and brands.
Monitor key business metrics through interactive dashboards.
Generate actionable insights for marketing and product strategy.

Compared product performance across multiple categories.
